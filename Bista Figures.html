We will work with genus level data. For this study genera and species were quite similar in their diversity patterns for this group. Here we used genera as it was a more reliable taxonomic assignment level for the metabarcoding data.

Convert OTU (eDNA) tables to genera tables (combine species in the same genera)

OTUList_water<-OTUList[OTUList$OTU%in%colnames(Water_Hel)[6:ncol(Water_Hel)],]
OTUList_water<-OTUList_water[OTUList_water$Genus!="G",]

WaterGenera<-data.frame(matrix(ncol=length(unique(OTUList_water$Genus)),nrow=length(Water$SamplingID),dimnames=list(c(as.character(Water$SamplingID)),c(as.character(unique(OTUList_water$Genus))))))


for(i in colnames(WaterGenera)){
  for(j in rownames(WaterGenera)){
  
  Water[Water$SamplingID==j,which(colnames(Water)%in%OTUList_water[OTUList_water$Genus==i,"OTU"])]
    
    
    if(sum(Water[Water$SamplingID==j,which(colnames(Water)%in%OTUList_water[OTUList_water$Genus==i,"OTU"])])==0){WaterGenera[j,i]<-0}
    else if(sum(Water[Water$SamplingID==j,which(colnames(Water)%in%OTUList_water[OTUList_water$Genus==i,"OTU"])])>=1){WaterGenera[j,i]<-1}
    else{print("cat")}
    
  }
}

WaterGenera<-WaterGenera[,colnames(WaterGenera)[colnames(WaterGenera)!="G"]]

The code is not perfect and uses rbind to grow a data frame. This can lead to artificial growing if you are not careful with your script so it is a good idea to check your final length after each loop.

statistical modeling (OTU richness)

cubic models first for the exuvia OTUs second for the water OTUs

#exuvia otu diversity modeled over time

#subset the data (easier for coding the lm function)

x1e<-OTU_mod[OTU_mod$Method=="Exuviae OTU","Time"]
ye<-OTU_mod[OTU_mod$Method=="Exuviae OTU","Alpha"]
c1e<-lm(ye~poly(x1e,3))

#check the stats
anova(c1e)

#get the r square value
summary(c1e)$r.squared

#water (eDNA) otu diversity modeled over time

x1w<-OTU_mod[OTU_mod$Method=="Water OTU","Time"]
yw<-OTU_mod[OTU_mod$Method=="Water OTU","Alpha"]
c1w<-lm(yw~poly(x1w,3))

#check the stats
anova(c1w)

#get the r square value
summary(c1w)$r.squared

OTUpredictsw<-data.frame(fits=predict(c1w,newdata=data.frame(x1w=seq(from=1,to=340,by=1))),

#water, exuvia genera level

Gen_mod<-data.frame(Alpha=AlphaTable[AlphaTable$var=="Exuviae"|AlphaTable$var=="Water"|AlphaTable$var=="Taxon","value"],
                    Method=AlphaTable[AlphaTable$var=="Exuviae"|AlphaTable$var=="Water"|AlphaTable$var=="Taxon","var"],
                    Time= AlphaTable[AlphaTable$var=="Exuviae"|AlphaTable$var=="Water"|AlphaTable$var=="Taxon","t_day"])


#cubic polynomial for the exuvia genea data
